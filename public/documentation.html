myFlix - Movies RESTful API

This web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints and how to access

Request URL HTTP Method Request body data format Response body data format

Allow users to update own user information, e.g. password. /users/[user ID]/[new password] PUT N/A Text message indicating if information was successfully updated.
Allow users to add movies to own favourites lists. /users/[user ID]/favourites/[movie title] POST N/A Text message indicating if movie was successfully added to favourites list.
Allow users to delete movies from own favourites lists. /users/[user ID]/favourites/[movie title] DELETE N/A Text message indicating if movie was successfully deleted from favourites list.
Allow users to deregister & delete own data from database. /users/[user ID] DELETE N/A Text message indicating if user was successfully deregistered & deleted from database.
